Welcome to Music-Collection!!!


This application maintains two types of users in its database, the common user and the administrators .

It queries an API that returns the artist's name and artist's Twitter @.

The common user should be able to register and change albums for available artists.

Administrators can do everything that regular users can and additionally, they can delete albums.
Administrators can also make a user an administrator
